# 多语言支持
- 项目默认只支持中文和英文
- 后台、网站、移动端默认根据用户本地语言展示，没有的将以中文展示

## 变量

```js
//##vue
//lang.json
{
    "introTips": "I am {name}.I am from {region}"
}
//使用
{{ $t('introTips', { name: 'Gopal一号', region: 'China' }) }}
// ##laravel
//lang.json
{
    "introTips": "I am :name.I am from :region"
}
//使用
__('introTips',['name'=>'Gopal一号','region'=>'China'])
```
