# 常用开发工具

  • window10系统
  • Laravel Homestead(免费) 为了快速搭建window10下的linux+laravel环境（当然你也可以用phpStudy，这里使用linux虚拟机，为的是熟悉linux的命令行，你可以使用xshell像连接服务器的方式连接本地的linux服务器）
  • phpStorm编辑器(收费)下载后，需要安装插件（laravel相关插件、VUE相关插件）
  • Windows PowerShell(window10自带的shell工具，在需要使用的目录，shift+右击就可以找到)
  • Xshell+Xftp连接linux服务器工具
  • HBuilder x 用来开发小程序端
  • Postman api接口测试工具
  • Google Chrome

# DSSHOP使用

  • 下载dsshop
  • 解压后，目录中主要分为admin(后端laravel代码部分)和api(vue前端代码部分)
  • 下面会分别介绍前端代码和后端代码的部属

# DSSHOP后端代码部署

  • 进入api目录
  • 安装composer 建议切换到中国镜像，不然下载会比较慢
  • 安装dsshop依赖包
composer install
php artisan migrate
php artisan generate:demo
#运行中可能会报错，如果是黄色信息可跳过，不引响正常使用，红色报错需要修复后才能继续

# DSSHOP前端代码部署(后台)

  • 进入admin目录
  • 安装node.js
  • 唤出Windows PowerShell，输入以下命令
# 安装依赖
npm install

# 建议不要用 cnpm 安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 修改config下的dev.env.js中的BASE_API
BASE_API: '"[配置的域名，区分http和https]/api/v1/admin/"'

# 本地开发 启动项目
npm run dev

# DSSHOP前端代码部署(H5端、小程序端)

  • 先安装nodejs
  • 打开HBuilder x，导入项目->选择'项目根目录/trade/Dsshop'
  • 修改App.vue
//33行
var applySecret = '', host = "【自己的域名，区分http和https】/api/v1/app/" // 服务器后端地址
//41行
secret: '' //项目根目录/api/.env中APP_KEY的值
