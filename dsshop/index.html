DSHOP电商商城

前后端分离架构，VUE2.0+Laravel7，免费开源可商用。

快速上手 →

优雅

API基于Larave，它是优雅的 PHP Web 开发框架。具有高效、简洁、富于表达力等优点。

最佳实践

合理的框架选择，良好的工程实践助你持续产出高质量代码。

跨平台

商城前端采用uni-app，一套代码编到10个平台，这不是梦想。

前后端分离

前后端分离，Laravel负责RSET API 服务，vue负责前端的展示和交互。

用户、角色、权限

内置RBAC控制权限模块，全程界面配置。

稳定更新

定期发行，向上兼容，平滑更新，减少因更新造成的不必要开支。
